# PWA 图标说明

此目录需要添加以下图标文件：

1. `icon-192x192.png` - 192×192 像素的应用图标
2. `icon-512x512.png` - 512×512 像素的应用图标

## 建议的图标设计

- 主色：靛蓝 (#4F46E5)
- 背景：白色或浅蓝
- 图标内容：字母 "S" 或 耳机/播放图标
- 风格：简洁现代，适合移动端

## 如何生成图标

可以使用以下工具生成 PWA 图标：
- https://realfavicongenerator.net/
- https://favicon.io/
- https://www.pwabuilder.com/imageGenerator

## 临时方案

在添加真实图标之前，PWA 功能仍然可以正常工作，只是会使用浏览器默认图标。
