# ShadowEnglish 需求文档

## 1. 音频播放器（AudioPlayer）- P0
### 1.1 音频来源
- URL 输入：支持粘贴播客/视频链接，自动提取音频地址
- 本地文件：支持上传 MP3 文件

### 1.2 播放控制
- 播放/暂停（空格键快捷键）
- 停止（归零）
- 快退 5 秒/快进 5 秒（方向键快捷键）
- 进度条（可拖拽）
- 音量滑块

### 1.3 影子训练功能
- 变速播放：0.5x / 0.75x / 1.0x / 1.25x / 1.5x（变速不变调）
- AB 复读：A 键设置 A 点，B 键设置 B 点，C 键清除

## 2. 字幕显示与单词卡 - P1
### 2.1 文本来源
- 手动粘贴文本模式
- 单词悬停翻译卡片

## 3. 听写练习 - P2
### 3.1 功能
- 隐藏字幕，用户输入听到的内容
- 提交后与原文对比，高亮差异

## 4. 跟读录音与评估 - P3
### 4.1 录音功能
- 使用 MediaRecorder API 录音
- 实时音频波形可视化

### 4.2 发音评估
- 基础模式：Web Speech API
- 专业模式：Google Cloud Speech-to-Text

## 5. 生词本 - P4
### 5.1 功能
- IndexedDB 存储
- 单词详情卡片
- 导出 CSV

## 6. 设置页 - P5
### 6.1 设置项
- OpenAI API Key（Whisper 转录）
- 翻译服务选择
- 发音评估档位
- 主题选择
- 字幕字体大小

## 7. PWA 功能
- 添加到主屏幕
- 离线缓存
- 快捷方式

## 8. UI/UX 需求
### 8.1 布局
- 移动端：单列布局
- 平板（≥768px）：两栏布局

### 8.2 导航
- 底部导航栏：播放器 | 听写 | 跟读 | 生词本 | 设置

### 8.3 配色
- 主色：靛蓝 (#4F46E5)
- 辅助色：橙色 (#F59E0B)

### 8.4 触控优化
- 所有可交互按钮最小尺寸 44×44px
