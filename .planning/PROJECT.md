# ShadowEnglish - 英语听说影子训练 PWA 应用

## 项目概述
构建一款面向英语学习者的「影子跟读训练工具」，以 Progressive Web App（PWA）形式交付，需适配 Android 手机与 iPad 平板，支持离线缓存部分功能。

## 技术栈
- **框架**: React + Vite
- **样式**: Tailwind CSS
- **PWA**: vite-plugin-pwa
- **数据存储**: IndexedDB (idb 库)
- **音频处理**: Web Audio API
- **语音识别**: Web Speech API

## 核心功能模块
1. 音频播放器（变速播放、AB 复读）
2. 字幕/文本显示与翻译
3. 听写练习模式
4. 跟读录音与评估
5. 生词本
6. 设置页

## 目标用户
英语学习者，需要进行听说训练的用户。
